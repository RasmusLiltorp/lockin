### Question 1 â€” Recurrence solving via the Master Theorem (asymptotic Theta bound)
**Question.** Which of the following answers holds for the following recurrence equation (rekursionsligning)? $T(n) = 3 \cdot T(n/2) + n^2$

**Options.**
0. $T(n) = \Theta(\log n)$
1. $T(n) = \Theta(\sqrt{n})$
2. $T(n) = \Theta(n)$
3. $T(n) = \Theta(n \log n)$
4. **$T(n) = \Theta(n^2)$ âś“**
5. $T(n) = \Theta(n^2 \log n)$
6. $T(n) = \Theta(n^3)$
7. The recurrence cannot be solved with the Master Theorem.

**Answer.** alternative 4: $T(n) = \Theta(n^2)$

**Method (reusable).** Read off $a$, $b$, $f(n)$ from $T(n)=a\cdot T(n/b)+f(n)$. Compute the critical exponent $c=\log_b a$ and the watershed $n^c$. Compare $f(n)$ to $n^c$: if $f$ grows polynomially faster, Case 3 gives $\Theta(f)$; slower, Case 1 gives $\Theta(n^c)$; same order, Case 2 gives $\Theta(n^c \log n)$. For Case 3 confirm regularity $a\cdot f(n/b)\le k\cdot f(n)$ for some $k<1$.

**Worked solution.** $a=3$, $b=2$, $f(n)=n^2$. Critical exponent $c=\log_2 3\approx 1.585$, watershed $n^{1.585}$. Here $f(n)=n^2=n^{1.585+0.415}$, so $f$ grows polynomially faster ($\varepsilon\approx 0.415>0$): Case 3. Regularity: $a\cdot f(n/b)=3\cdot(n/2)^2=\tfrac34 n^2$ with $k=\tfrac34<1$ holds. So $T(n)=\Theta(f(n))=\Theta(n^2)$.

### Question 2 â€” Recurrence solving via Master Theorem
**Question.** Which of the following holds for the recurrence relation $T(n) = 9 \cdot T(n/3) + n^{1/2}$?

**Options.**
0. $T(n)=\Theta(\log n)$
1. $T(n)=\Theta(\sqrt{n})$
2. $T(n)=\Theta(n)$
3. $T(n)=\Theta(n\log n)$
4. **$T(n)=\Theta(n^2)$ âś“**
5. $T(n)=\Theta(n^2\log n)$
6. $T(n)=\Theta(n^3)$
7. The recurrence cannot be solved with the Master Theorem.

**Answer.** alternative 4: $T(n)=\Theta(n^2)$

**Method (reusable).** Identify $a$, $b$, $f(n)$. Compute the watershed $n^{\log_b a}$. If $f(n)=O(n^{\log_b a-\varepsilon})$, Case 1 gives $\Theta(n^{\log_b a})$; if equal up to logs, Case 2; if $\Omega(n^{\log_b a+\varepsilon})$ with regularity, Case 3 gives $\Theta(f(n))$.

**Worked solution.** $a=9$, $b=3$, $f(n)=n^{1/2}$. $\log_b a=\log_3 9=2$, watershed $n^2$. $f(n)=n^{0.5}$ grows polynomially slower than $n^2$ ($n^{0.5}=O(n^{2-\varepsilon})$ for any $\varepsilon<1.5$): Case 1. So $T(n)=\Theta(n^2)$.

### Question 3 â€” Solve a recurrence with the Master Theorem

### Question 4 â€” Recurrence solving via Master Theorem
**Question.** Which of the following holds for the recurrence (rekursionsligning) $T(n) = 2 \cdot T(n/3) + n \log n$?

**Options.**
0. $T(n) = \Theta(\log n)$
1. $T(n) = \Theta(\sqrt{n})$
2. $T(n) = \Theta(n)$
3. **$T(n) = \Theta(n \log n)$ âś“**
4. $T(n) = \Theta(n^2)$
5. $T(n) = \Theta(n^2 \log n)$
6. The recurrence cannot be solved with the Master Theorem (kan ikke lĂ¸ses med Master Theorem).

**Answer.** alternative 3: $T(n) = \Theta(n \log n)$

**Method (reusable).** Apply the Master Theorem: compute $c=\log_b a$, compare $f(n)$ to $n^c$, and check the regularity condition for Case 3.

**Worked solution.** $a=2$, $b=3$, $f(n)=n\log n$. Critical exponent $c=\log_3 2\approx 0.631$, so $n^c\approx n^{0.63}$. $f(n)=n\log n$ grows polynomially faster than $n^{0.63}$ (by factor $\approx n^{0.37}\log n$), so $f(n)=\Omega(n^{c+\varepsilon})$: Case 3. Regularity: $a\cdot f(n/b)=2\cdot(n/3)\log(n/3)=\tfrac23 n\log(n/3)\le d\cdot n\log n$ with $d=\tfrac23<1$ for large $n$. So $T(n)=\Theta(f(n))=\Theta(n\log n)$.

### Question 5 â€” Asymptotic notation (Big-O) truth-evaluation; multiple correct answers
**Question.** Which of the following statements are true? As usual, $\log n$ denotes the logarithm with base two. [One or more answers.]

**Options.**
0. $n$ is $O(\log n)$
1. **$(\log n)^3$ is $O(n^2)$ âś“**
2. **$n\log n$ is $O(n^{1.5})$ âś“**
3. $2^n$ is $O(\sqrt{n})$
4. **$3n^2$ is $O(n^2)$ âś“**
5. $7^n$ is $O((\log n)^7)$
6. **$\log(n!)$ is $O(n^2)$ âś“**

**Answer.** alternatives 1, 2, 4, 6 (multiple correct): $(\log n)^3$ is $O(n^2)$; $n\log n$ is $O(n^{1.5})$; $3n^2$ is $O(n^2)$; $\log(n!)$ is $O(n^2)$.

**Method (reusable).** For each "$f$ is $O(g)$" claim, $f$ is $O(g)$ iff $f/g$ stays bounded as $n\to\infty$. Use the growth order: constants $<\log n<$ polylog $(\log n)^k<n^\varepsilon<$ polynomial $n^c<$ exponential $c^n$. Polylog is dominated by any positive power of $n$. Constant factors are absorbed by $O$. Use $\log(n!)=\Theta(n\log n)$ (Stirling).

**Worked solution.** [0] FALSE: $n$ grows faster than $\log n$. [1] TRUE: polylog dominated by any polynomial; $(\log n)^3/n^2\to 0$. [2] TRUE: equivalent to $\log n=O(n^{0.5})$; ratio $\log n/\sqrt n\to 0$. [3] FALSE: exponential dominates any power. [4] TRUE: constant factor 3 absorbed. [5] FALSE: exponential dominates polylog. [6] TRUE: $\log(n!)=\Theta(n\log n)$, below $n^2$. Correct set: 1, 2, 4, 6.

### Question 6 â€” Asymptotic notation: O / o / Theta / Omega / omega claims (multiple correct)

**Options.**
0. $\log n$ is $\omega(n^2)$
1. **$n^2 + n^3$ is $\Theta(n^3)$ âś“**
2. $6$ is $o(7)$
3. **$3^n$ is $\Omega(2^n)$ âś“**
4. $n/(\log n)^2$ is $o((\log n)^3)$
5. **$n^n$ is $\Omega(2^n)$ âś“**
6. **$n^{1.1}$ is $\omega(n \log n)$ âś“**

**Answer.** alternatives 1, 3, 5, 6 (multiple correct): $n^2+n^3$ is $\Theta(n^3)$; $3^n$ is $\Omega(2^n)$; $n^n$ is $\Omega(2^n)$; $n^{1.1}$ is $\omega(n\log n)$.

**Method (reusable).** For each claim compute $L=\lim f(n)/g(n)$. Then: $o$ true iff $L=0$; $\omega$ true iff $L=\infty$; $\Theta$ true iff $0<L<\infty$; $\Omega$ true iff $L>0$ (including $\infty$); $O$ true iff $L<\infty$. **Worked solution.** [0] FALSE: $\log n/n^2\to 0$, so $\log n$ is $o(n^2)$, not $\omega$. [1] TRUE: dominated by $n^3$, ratio $\to 1$. [5] TRUE: $n^n/2^n\to\infty$. [6] TRUE: $n^{1.1}/(n\log n)=n^{0.1}/\log n\to\infty$. Correct set: 1, 3, 5, 6.

### Question 7 â€” Binary max-heap property verification (multiple correct)

**Answer.** alternatives 0, 1, 4, 6 (multiple correct). Each satisfies $A[i]\ge A[2i]$ and $A[i]\ge A[2i+1]$ for every node with children.

**Method (reusable).** Treat $A$ as a 1-indexed complete binary tree: node $i$ has children $2i$ and $2i+1$. For each $i$ from 1 to $n$, check every existing child $c\le n$: max-heap needs $A[i]\ge A[c]$. If any parent is smaller than a child, it is not a max-heap.

**Worked solution.** Alt 0 $[2,1]$: $2\ge 1$ OK. Alt 1 $[2,1,2,1]$: node 1 children $1,2$ ($2\ge 1$, $2\ge 2$); node 2 child idx4=1 ($1\ge 1$) OK. Alt 6 $[1,-1,-1,-2,-2,-2]$: node 1 $\ge -1,-1$; node 2 $\ge -2,-2$; node 3 $\ge -2$ OK. Correct: 0, 1, 4, 6.

### Question 8 â€” MAX-HEAP-INSERT sequence (CLRS)
**Question.** Insert into a binary max-heap $A$ the elements 6, 12, 13, 4, 8, 14 and 5 in the given order using MAX-HEAP-INSERT. The heap $A$ is empty to start with. Which of the answers below shows the appearance of $A$ afterwards?

**Options.**
0. $A=[14, 12, 13, 4, 8, 6, 5]$ (indices 1-7)
1. $A=[6, 12, 13, 4, 8, 14, 5]$ (indices 1-7)
2. **$A=[14, 8, 13, 4, 6, 12, 5]$ (indices 1-7) âś“**
3. $A=[14, 13, 12, 8, 6, 5, 4]$ (indices 1-7)
4. $A=[13, 12, 14, 4, 8, 6, 5]$ (indices 1-7)

**Answer.** alternative 2: $A=[14, 8, 13, 4, 6, 12, 5]$

**Method (reusable).** Start with an empty 1-indexed array heap. For each element in order: append at the next free leaf, then sift up â€” while the node exceeds its parent, swap with the parent; stop at the root or when parent $\ge$ node. This is insertion order, not a sort of the multiset, so the result is usually not fully sorted.

**Worked solution.** Insert 6: $[6]$. Insert 12: $[6,12]\to[12,6]$. Insert 13: $[12,6,13]\to[13,6,12]$. Insert 4: $[13,6,12,4]$ (4<6 stop). Insert 8: $[13,6,12,4,8]$, parent idx2=6, 8>6 swap $\to[13,8,12,4,6]$ (8<13 stop). Insert 14: $[13,8,12,4,6,14]$, parent idx3=12, swap $\to[13,8,14,4,6,12]$, 14>13 swap $\to[14,8,13,4,6,12]$. Insert 5: $[14,8,13,4,6,12,5]$, parent idx3=13, 5<13 stop. Final $[14,8,13,4,6,12,5]$.

### Question 11 â€” CLRS Lomuto PARTITION trace on a subarray
**Question.** Execute PARTITION(A,4,13) on the relevant part of the array below. Array $A$ (indices 1..15): $[21,17,28,14,9,18,6,1,26,15,30,7,13,19,2]$. Which of the following shows the appearance of $A$ afterward?

**Options.**
0. $A = [21,17,28,1,6,7,9,13,14,15,18,26,30,19,2]$
1. $A = [1,2,6,7,9,13,14,15,17,18,19,21,26,28,30]$
2. $A = [21,17,28,9,6,1,7,13,14,18,26,15,30,19,2]$
3. **$A = [21,17,28,9,6,1,7,13,26,15,30,14,18,19,2]$ âś“**

**Answer.** alternative 3: $A = [21,17,28,9,6,1,7,13,26,15,30,14,18,19,2]$

**Method (reusable).** CLRS Lomuto PARTITION$(A,p,r)$: pivot $x=A[r]$ (last element of the range). Set $i=p-1$. For $j=p$ to $r-1$: if $A[j]\le x$, increment $i$ and swap $A[i]\leftrightarrow A[j]$. Finally swap $A[i+1]\leftrightarrow A[r]$. Only indices $p..r$ change.

After the $j$-loop and final pivot swap, range 4..13 becomes $9,6,1,7,13,26,15,30,14,18$. Splicing with the untouched ends: $A=21,17,28,\;9,6,1,7,13,26,15,30,14,18,\;19,2$.

**Answer.** alternative 1: 2

**Method (reusable).** Build the Huffman tree by greedily merging the two lowest-weight nodes. The codeword length of a symbol is its leaf depth. Report that depth.
