### Question 1 â€” Asymptotic solution of a divide-and-conquer recurrence via the Master Theorem.
**Question.** Which of the following answers holds for the following recurrence (rekursionsligning)? $T(n) = T(n/4) + n$
**Options.**
0. $T(n)=\Theta(\log n)$
1. $T(n)=\Theta(n^{1/4})$
2. **$T(n)=\Theta(n)$ âœ“**
3. $T(n)=\Theta(n\log n)$
4. $T(n)=\Theta(n^{4})$
5. The recurrence cannot be solved with the Master Theorem (kan ikke lÃ¸ses med Master Theorem).
**Answer.** alternative 2: $T(n)=\Theta(n)$
**Method (reusable).** 1. Write recurrence as $T(n)=aT(n/b)+f(n)$; identify $a,b,f$. 2. Compute watershed $n^{\log_b a}$. 3. Compare $f$ to watershed for the three Master Theorem cases. 4. Match to option.
**Worked solution.** $a=1$, $b=4$, $f(n)=n$. $\log_b a = \log_4 1 = 0$, so $n^{\log_b a}=n^0=1$. $f(n)=n=\Omega(n^{0+1})$, polynomially larger than $1$, so case 3. Regularity: $a f(n/b)=n/4 \le c n$ with $c=1/4<1$ holds. Therefore $T(n)=\Theta(f(n))=\Theta(n)$. Intuition: cost is dominated by the top level $n$, the geometric sum $n + n/4 + n/16 + \dots = (4/3)n = \Theta(n)$.

### Question 2 â€” Master Theorem application to a divide-and-conquer recurrence
**Question.** Which of the following answers holds for the following recurrence (rekursionsligning)? $T(n) = 3T(n/4) + n$
**Options.**
0. $T(n) = \Theta(\log n)$
1. $T(n) = \Theta(n^{3/4})$
2. $T(n) = \Theta(n^{\alpha})$ with $\alpha = \log_4(3)$
3. **$T(n) = \Theta(n)$ âœ“**
4. $T(n) = \Theta(n \log n)$
5. $T(n) = \Theta(n^{4/3})$
6. The recurrence cannot be solved with the Master Theorem.
**Answer.** alternative 3: $T(n) = \Theta(n)$
**Method (reusable).** 1. Identify $a, b, f(n)$ from $T(n)=aT(n/b)+f(n)$. 2. Compute the critical exponent $c = \log_b(a)$. 3. Compare $f(n)$ with $n^c$: case 1 if $f$ smaller polynomially ($\Theta(n^c)$); case 2 if equal (extra log factor); case 3 if $f$ larger polynomially and regularity holds ($\Theta(f)$). 4. Report the resulting bound.
**Worked solution.** $a=3$, $b=4$, $f(n)=n$. Critical exponent $c = \log_4(3) \approx 0.792$. Compare $f(n)=n=n^1$ with $n^{0.792}$: $f$ is polynomially larger ($n = \Omega(n^{c+\varepsilon})$ for small $\varepsilon$). This is case 3. Regularity check: $a\cdot f(n/b)=3\cdot(n/4)=0.75n \le c\cdot n$ with $c=0.75<1$ holds. So $T(n)=\Theta(f(n))=\Theta(n)$.

### Question 3 â€” Algorithm analysis: solving a divide-and-conquer recurrence via the Master Theorem.
**Question.** Which of the following answers holds for the following recurrence equation (rekursionsligning)? $T(n) = 3T(n/4) + n^{3/4}$
**Options.**
0. $T(n)=\Theta(\log n)$.
1. $T(n)=\Theta(n^{3/4})$.
2. **$T(n)=\Theta(n^{\alpha})$ med (with) $\alpha=\log_4(3)$. âœ“**
3. $T(n)=\Theta(n)$.
4. $T(n)=\Theta(n\log n)$.
5. $T(n)=\Theta(n^{4/3})$.
6. The recurrence equation cannot be solved with the Master Theorem (kan ikke lÃ¸ses med Master Theorem).
**Answer.** alternative 2: $T(n)=\Theta(n^{\alpha})$ with $\alpha=\log_4(3)$.
**Method (reusable).** 1. Read off $a, b, f(n)$ from $T(n)=aT(n/b)+f(n)$. 2. Compute the critical exponent $c = \log_b(a)$. 3. Compare $f(n)=n^d$ with $n^c$. 4. If $d<c$: case 1, $T(n)=\Theta(n^c)$. If $d=c$: case 2, $T(n)=\Theta(n^c \log n)$. If $d>c$ (with regularity): case 3, $T(n)=\Theta(f(n))$. 5. Match to the option.
**Worked solution.** $a=3$, $b=4$, $f(n)=n^{3/4}$ so $d=0.75$. Critical exponent $c=\log_4(3)\approx 0.7925$. Since $d=0.75 < c\approx 0.7925$, $f(n)=O(n^{c-\varepsilon})$, so case 1 applies. Therefore $T(n)=\Theta(n^{\log_4 3})$, which is alternative 2.

### Question 4 â€” Asymptotic analysis of a recurrence (Master Theorem applicability).
**Question.** Which of the following holds for the recurrence (rekursionsligning) $T(n) = 3 T(n^{1/4}) + n^{3/4}$?
**Options.**
0. $T(n) = \Theta(\log n)$.
1. $T(n) = \Theta(n^{\alpha})$ with $\alpha = \log_4(3)$.
2. $T(n) = \Theta(n)$.
3. $T(n) = \Theta(n \log n)$.
4. $T(n) = \Theta(n^{4/3})$.
5. **The recurrence cannot be solved with the Master Theorem (Rekursionsligningen kan ikke loses med Master Theorem). âœ“**
**Answer.** alternative 5: The recurrence cannot be solved with the Master Theorem.
**Method (reusable).** 1. Check the form: the Master Theorem requires $T(n)=a T(n/b)+f(n)$ with the recursive argument being $n$ divided by a constant $b$. 2. If the argument is a root ($n^{1/4}$) or otherwise not $n/b$, substitute $n=2^m$ and $S(m)=T(2^m)$. 3. Rewrite: here $S(m)=3 S(m/4)+2^{(3/4)m}$. 4. For the standard Master Theorem on $S(m)$ the driving term must be polynomial in $m$, $\Theta(m^c)$. 5. If the driving term is exponential in $m$ (as $2^{(3/4)m}$ is), the standard Master Theorem does not apply. 6. Match against options: if no listed $\Theta$ bound equals the true growth, the answer is that the Master Theorem cannot be applied.
**Worked solution.** The recurrence $T(n)=3T(n^{1/4})+n^{3/4}$ has recursive argument $n^{1/4}$, not $n/b$ for a constant $b$, so it is not directly in Master Theorem form. Substituting $n=2^m$, $S(m)=T(2^m)$: $T(n^{1/4})=S(m/4)$ and $n^{3/4}=2^{(3/4)m}$, giving $S(m)=3S(m/4)+2^{(3/4)m}$. The standard Master Theorem compares the driving term against $m^{\log_4 3}$, but $2^{(3/4)m}$ is exponential in $m$, not polynomial, so the standard Master Theorem does not apply. (The recursion-tree sum is dominated by the root term, giving true growth $\Theta(n^{3/4})$, which is not among the listed $\Theta$ options anyway.) Correct alternative: 5.

### Question 5 â€” Multiple-answer: judging truth of asymptotic-growth statements (O, o, Theta, omega).
**Question.** Which of the following statements are true? [One or more answers.] (Asymptotic notation: $O$, $o$, $\Theta$, $\omega$.)
**Options.**
0. **$n^2$ is $O(n^3)$ âœ“**
1. **$n^2$ is $o(n^3)$ âœ“**
2. $n^2$ is $\Theta(3n^2+2n^3)$
3. $2^n$ is $O(n^4)$
4. **$n^2$ is $O(4^n)$ âœ“**
5. **$(\log n)^4$ is $O(n/(\log n)^4)$ âœ“**
6. **$4^n$ is $\omega(2^n)$ âœ“**
7. $(1/2)^n$ is $O((1/4)^n)$
8. **$1/\log n$ is $O(1)$ âœ“**
9. **$\sin n$ is $O(\log n)$ âœ“**
**Answer.** alternative 0 (lowest correct): true statements are 0, 1, 4, 5, 6, 8, 9. False: 2 ($\Theta$ fails, $2n^3$ dominates), 3 (exp beats poly), 7 (ratio $2^n\to\infty$).
**Method (reusable).** 1. For $O(g)$: check $f/g$ bounded as $n$ grows (limit finite). 2. For $o(g)$: check $f/g \to 0$. 3. For $\Theta(g)$: check $f/g$ bounded above AND below by positive constants. 4. For $\omega(g)$: check $f/g \to \infty$. 5. Use growth hierarchy: const $<$ $1/\log$ $<$ $\log^k$ $<$ polynomial $<$ exponential. 6. For exponentials $a^n$ vs $b^n$ compare bases. 7. Bounded functions (sin) are $O$ of anything that does not $\to 0$.
**Worked solution.** Computed ratios at large $n$. 0/1: $n^2/n^3=1/n\to 0$, so $O$ and $o$ hold. 2: $3n^2+2n^3$ dominated by $n^3$; $n^2/(2n^3)\to 0$, not bounded below, $\Theta$ fails. 3: $2^n/n^4\to\infty$, false. 4: $n^2/4^n\to 0$, $O$ holds. 5: ratio $= (\log n)^8/n \to 0$, $O$ holds. 6: $4^n/2^n=2^n\to\infty$, $\omega$ holds. 7: $(1/2)^n/(1/4)^n=2^n\to\infty$, not bounded, false. 8: $1/\log n\to 0$, bounded, $O(1)$ holds. 9: $|\sin n|\le 1\le \log n$ eventually, $O$ holds. True set $= \{0,1,4,5,6,8,9\}$.
