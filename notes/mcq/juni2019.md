### Question 1 ÔÇö Recurrence solving via Master Theorem (asymptotic complexity).
**Question.** Which of the answers below holds for the following recurrence (rekursionsligning)? $T(n)=T(3n/4)+n$
**Options.**
0. $T(n)=\Theta(1)$
1. $T(n)=\Theta(\log n)$
2. $T(n)=\Theta(n^{1/4})$
3. $T(n)=\Theta(n^{1/3})$
4. $T(n)=\Theta(n^{3/4})$
5. **$T(n)=\Theta(n)$ Ôťô**
6. $T(n)=\Theta(n\log n)$
7. $T(n)=\Theta(n^{\log_3 4})$
8. The recurrence cannot be solved with the Master Theorem (Rekursionsligningen kan ikke loses med Master Theorem).
**Answer.** alternative 5: $T(n)=\Theta(n)$
**Method (reusable).** 1. Write the recurrence as $T(n)=a\cdot T(n/b)+f(n)$; identify $a$, $b$, $f(n)$. 2. Compute $c=\log_b a$ and $n^c$. 3. Compare $f(n)$ to $n^c$ (case 1/2/3). 4. Match alternative.
**Worked solution.** Here $a=1$, $b=4/3$, $f(n)=n$. Critical exponent $c=\log_{4/3} 1 = 0$, so $n^c = n^0 = 1$. $f(n)=n$ is polynomially larger than $n^0$ (by $n^{1-0}$). Regularity: $a\cdot f(n/b)=1\cdot(3n/4)=3n/4 = (3/4)\cdot f(n)$, with constant $3/4<1$, so it holds. Master Theorem case 3 gives $T(n)=\Theta(f(n))=\Theta(n)$. This is alternative 5.

### Question 2 ÔÇö Recurrence solving via Master Theorem (asymptotic analysis)
**Question.** Which of the following answers holds for the following recurrence equation (rekursionsligning)? $T(n) = 3 \cdot T(n/3) + n$
**Options.**
0. $T(n)=\Theta(1)$
1. $T(n)=\Theta(\log n)$
2. $T(n)=\Theta(n^{1/3})$
3. $T(n)=\Theta(n)$
4. **$T(n)=\Theta(n\log n)$ Ôťô**
5. $T(n)=\Theta(n^3)$
6. The recurrence equation cannot be solved with the Master Theorem.
**Answer.** alternative 4: $T(n)=\Theta(n\log n)$
**Method (reusable).** 1. Identify $a$, $b$, $f(n)$ from $T(n)=a\cdot T(n/b)+f(n)$. 2. Compute the critical exponent $c = \log_b(a)$. 3. Compare $f(n)$ with $n^c$. 4. If $f(n)=\Theta(n^c)$ -> case 2 -> $T(n)=\Theta(n^c \log n)$; if $f(n)=O(n^{c-\varepsilon})$ -> case 1 -> $\Theta(n^c)$; if $f(n)=\Omega(n^{c+\varepsilon})$ with regularity -> case 3 -> $\Theta(f(n))$. 5. Match to options.
**Worked solution.** $a=3$, $b=3$, $f(n)=n$. Critical exponent $c = \log_3(3) = 1$, so $n^c = n$. $f(n)=n = \Theta(n^1) = \Theta(n^c)$. This is Master Theorem case 2, giving $T(n) = \Theta(n^c \log n) = \Theta(n \log n)$. Matches alternative 4.

### Question 3 ÔÇö Master Theorem applicability / asymptotic solution of a recurrence
**Question.** Which of the following answers holds for the following recurrence (rekursionsligning)? $T(n) = T(n/4) + \log n$.
**Options.**
0. $T(n)=\Theta(1)$
1. $T(n)=\Theta(\log n)$
2. $T(n)=\Theta(n^{1/4})$
3. $T(n)=\Theta(n)$
4. $T(n)=\Theta(n\log n)$
5. $T(n)=\Theta(n^4)$
6. **The recurrence cannot be solved with the Master Theorem (kan ikke loses med Master Theorem). Ôťô**
**Answer.** alternative 6: The recurrence cannot be solved with the Master Theorem.
**Method (reusable).** 1. Write recurrence as $T(n)=a\,T(n/b)+f(n)$; read $a$, $b$, $f(n)$. 2. Compute the watershed $n^{\log_b a}$. 3. Compare $f(n)$ to $n^{\log_b a}$: Case 1 if $f$ is polynomially smaller, Case 2 if same order (possibly times polylog), Case 3 if polynomially larger AND regularity holds. 4. If $f(n)$ sits between the cases (faster than the watershed but not by a polynomial factor $n^\varepsilon$), no case applies -> Master Theorem cannot be used. 5. Pick the 'cannot be solved' option if no case fits.
**Worked solution.** $a=1$, $b=4$, $f(n)=\log n$. Watershed $n^{\log_b a}=n^{\log_4 1}=n^0=1$. $f(n)=\log n$ is asymptotically larger than 1, so Case 1 fails. It is not $\Theta(1)$ (with or without a polylog factor matching the watershed), so Case 2 does not match. For Case 3 we would need $f(n)=\Omega(n^\varepsilon)$ for some $\varepsilon>0$, but $\log n$ grows slower than any positive power $n^\varepsilon$, so $f(n)$ is not polynomially larger and Case 3 fails. $f(n)$ lies in the gap between Case 1 and Case 3, so the standard Master Theorem gives no case. (The true solution telescopes to $\Theta(\log^2 n)$, which is none of the options.) Therefore alternative 6.

### Question 4 ÔÇö Solving a recurrence via the Master Theorem (asymptotic complexity).
**Question.** Which of the following holds for the recurrence relation $T(n) = 4 \cdot T(n/3) + n\log n$?
**Options.**
0. $T(n)=\Theta(1)$
1. $T(n)=\Theta(\log n)$
2. $T(n)=\Theta(n^{1/4})$
3. $T(n)=\Theta(n^{1/3})$
4. $T(n)=\Theta(n^{3/4})$
5. $T(n)=\Theta(n^{\log_4 3})$
6. $T(n)=\Theta(n)$
7. $T(n)=\Theta(n\log n)$
8. **$T(n)=\Theta(n^{\log_3 4})$ Ôťô**
9. The recurrence cannot be solved with the Master Theorem (Rekursionsligningen kan ikke loses med Master Theorem).
**Answer.** alternative 8: $T(n)=\Theta(n^{\log_3 4})$
**Method (reusable).** 1. Identify $a$, $b$, $f(n)$ from $T(n)=a\cdot T(n/b)+f(n)$. 2. Compute the critical exponent $\log_b a$. 3. Compare $f(n)$ to $n^{\log_b a}$: if $f$ is polynomially smaller (Case 1), answer is $\Theta(n^{\log_b a})$; if same order up to polylog (Case 2), multiply by log; if polynomially larger and regularity holds (Case 3), answer is $\Theta(f(n))$. 4. Match to options.
**Worked solution.** $a=4$, $b=3$, $f(n)=n\log n$. $\log_b a = \log_3 4 \approx 1.262$. Since $n\log n = O(n^{1.262-\varepsilon})$ for small $\varepsilon>0$ ($n\log n$ grows slower than $n^{1.262}$), $f(n)$ is polynomially smaller than $n^{\log_3 4}$. This is Master Theorem Case 1, giving $T(n)=\Theta(n^{\log_3 4})$. Matches alternative 8.

### Question 5 ÔÇö Multiple-answer Big-O asymptotic comparison (true/false per statement).
**Question.** Which of the statements below are true? [One or more answers.] (5 points)
**Options.**
0. $n^3$ is $O(n^2)$
1. **$\log n$ is $O(n^{1/2})$ Ôťô**
2. **$1$ is $O(n^{1/3})$ Ôťô**
3. $n^{3/2}$ is $O(n\log n)$
4. $1.5^n$ is $O(n^{15})$
5. **$n\log n$ is $O(n(\log n)^3 + n^{1/3})$ Ôťô**
**Answer.** alternative 1: True statements are option 1 ($\log n$ is $O(n^{1/2})$), option 2 ($1$ is $O(n^{1/3})$), and option 5 ($n\log n$ is $O(n(\log n)^3 + n^{1/3})$). Options 0, 3, 4 are false.
**Method (reusable).** 1. For each claim "$f$ is $O(g)$", check whether $f(n)/g(n)$ stays bounded as $n$ grows. 2. Use the growth hierarchy: constants < logs < polynomials (by exponent) < exponentials. 3. $f$ is $O(g)$ iff $g$ grows at least as fast as $f$. 4. For exponential vs polynomial, compare logs ($n\ln(\text{base})$ vs $k\ln n$); exponential always wins, so exp is never $O(\text{poly})$. 5. For sums in $g$, the dominant term sets the bound.
**Worked solution.** 0: $n^3/n^2 = n \to \infty$, FALSE. 1: $\log n$ grows slower than any positive power, so $\log n = O(n^{1/2})$, ratio $\to 0$, TRUE. 2: constant 1 is $O$ of anything growing, $1/n^{1/3}\to 0$, TRUE. 3: $n^{1.5}/(n\log n) = n^{0.5}/\log n \to \infty$, FALSE. 4: $\log(1.5^n/n^{15})=n\ln 1.5 - 15\ln n \to +\infty$ (exponential beats polynomial), FALSE. 5: dominant term of $g$ is $n(\log n)^3$; $(n\log n)/(n(\log n)^3) = 1/(\log n)^2 \to 0$, bounded, TRUE. True set $=\{1,2,5\}$.

### Question 6 ÔÇö Asymptotic notation (Theta, Omega, little-o, little-omega) truth evaluation; multiple correct answers
**Question.** Which of the following statements are true? [One or more answers.] (Hvilke af nedenst├ąende udsagn er sande? [Et eller flere svar.])
**Options.**
0. **$n^{1/2} + 2n^2 + (\log n)^2$ is $\Theta(n^2)$ Ôťô**
1. **$n$ is $\Omega(n)$ Ôťô**
2. $2^n$ is $o(n^3)$
3. $(\log n)^3$ is $\omega(n / \log n)$
4. **$n^{1/3}$ is $o(n^{1/2})$ Ôťô**
5. $2n^3 + 4n^5$ is $\Theta(5n^4 + 3n^2)$
**Answer.** alternative 0: True statements are (0) $n^{1/2}+2n^2+(\log n)^2 = \Theta(n^2)$; (1) $n = \Omega(n)$; (4) $n^{1/3} = o(n^{1/2})$. Options 2, 3, 5 are false.
**Method (reusable).** 1. For each statement, find the dominant term of each side. 2. $\Theta(g)$: both sides same growth order. 3. $\Omega(g)$: LHS grows at least as fast as $g$. 4. $o(g)$: LHS grows strictly slower ($f/g \to 0$). 5. $\omega(g)$: LHS grows strictly faster ($f/g \to \infty$). 6. Hierarchy: polylog < polynomial < exponential.
**Worked solution.** Opt0: $\max(n^{1/2}, 2n^2, (\log n)^2) = 2n^2$, so the sum is $\Theta(n^2)$. TRUE. Opt1: $n$ grows exactly as $n$, so $n = \Omega(n)$. TRUE. Opt2: $2^n$ is exponential, dominates $n^3$, so it is NOT $o(n^3)$ (it is $\omega(n^3)$). FALSE. Opt3: $(\log n)^3$ is polylog; $n/\log n$ grows polynomially (faster than any polylog), so $(\log n)^3 = o(n/\log n)$, not $\omega$. FALSE. Opt4: exponents $1/3 < 1/2$, and $\lim n^{1/3}/n^{1/2} = \lim n^{-1/6} = 0$, so $n^{1/3} = o(n^{1/2})$. TRUE. Opt5: LHS dominant $4n^5$ (degree 5); RHS dominant $5n^4$ (degree 4). Different orders, not $\Theta$. FALSE. Correct $=\{0,1,4\}$.

### Question 8 ÔÇö Heap operation trace: one HEAP-EXTRACT-MAX on a max-heap (CLRS), array representation.
**Question.** Perform HEAP-EXTRACT-MAX(A) on the array A below. A (positions 1..9): [5, 4, 3, 3, 4, 2, 3, 2, 1]. Which of the following answers shows what A looks like afterwards?
**Options.**
0. A (1..9): [4, 4, 3, 3, 3, 2, 2, 1, (blank)]
1. A (1..9): [(blank), 4, 3, 3, 4, 2, 3, 2, 1]
2. A (1..9): [4, 3, 3, 4, 2, 3, 2, 1, (blank)]
3. **A (1..9): [4, 4, 3, 3, 1, 2, 3, 2, (blank)] Ôťô**
4. A (1..9): [4, 4, 3, 3, 2, 3, 2, 1, (blank)]
**Answer.** alternative 3: A (1..9): [4, 4, 3, 3, 1, 2, 3, 2, (blank)]
**Method (reusable).** 1. HEAP-EXTRACT-MAX: save $A[1]$ as the max. 2. Move the last element $A[n]$ into $A[1]$. 3. Decrement heap size by 1 (last slot blank). 4. Run MAX-HEAPIFY(A,1): compare the node with its two children (children of $i$ at $2i, 2i+1$), swap with the larger child if a child is bigger, recurse downward. 5. Read off the array.
**Worked solution.** max $= A[1] = 5$. Move $A[9]=1$ to $A[1]$; heap size $= 8$, slot 9 blank. Array: [1,4,3,3,4,2,3,2]. Pos 5 children would be 10,11 (out of heap, size 8), stop. Final: [4,4,3,3,1,2,3,2] with position 9 blank. Matches alternative 3.

### Question 9 ÔÇö Hashing with linear probing; find all auxiliary-hash start positions consistent with where the key landed. Multiple-answer.
**Question.** We have a hash table $H$ using linear probing and an auxiliary hash function $h'(x)$. The table currently looks like: index 0..6 with $H[0]=33$, $H[1]=$ empty, $H[2]=27$, $H[3]=32$, $H[4]=55$, $H[5]=$ empty, $H[6]=47$. Then 99 is inserted, after which the table is: $H[0]=33$, $H[1]=99$, $H[2]=27$, $H[3]=32$, $H[4]=55$, $H[5]=$ empty, $H[6]=47$. Which of the following values of $h'(99)$ are possible? (One or more answers.)
**Options.**

1. **$h'(99)=1$ Ôťô**
2. $h'(99)=2$
3. $h'(99)=3$
4. $h'(99)=4$
5. $h'(99)=5$
6. **$h'(99)=6$ Ôťô**
**Answer.** alternative 0: $h'(99)=0$, $h'(99)=1$, and $h'(99)=6$ are all possible (alternatives 0, 1, and 6). Multiple-answer.
**Method (reusable).** 1. With linear probing, insertion starts at slot $h'(x)$ and probes forward $(\text{slot}+1) \bmod m$ until the first empty slot, where the key is placed. 2. Note the table size $m$ and which slots were occupied BEFORE the insertion. 3. Note the slot where the new key ended up (landing slot $L$). 4. For each candidate start $s$ in $0..m-1$, simulate probing from $s$ over the pre-insertion table; record the first empty slot reached. 5. $h'(x)=s$ is possible iff its first empty slot equals $L$.

So $h'(99) \in \{0,1,6\}$.

### Question 10 ÔÇö Hashing / double hashing probe sequence ÔÇö find valid second-hash values (multiple correct)

### Question 11 ÔÇö Quicksort PARTITION trace (CLRS Lomuto partition, pivot = last element)
**Question.** Run PARTITION(A,1,7) on the array below. A (indices 1..7) = [6, 2, 4, 5, 1, 7, 3]. Which alternative shows the appearance of A afterwards? (Standard CLRS Lomuto partition; pivot = last element A[7]=3.)
**Options.**
0. **A = [2, 1, 3, 5, 6, 7, 4] Ôťô**
1. A = [2, 1, 3, 6, 4, 5, 7]
2. A = [1, 2, 3, 4, 5, 6, 7]
3. A = [3, 2, 1, 4, 5, 6, 7]
4. A = [1, 2, 3, 7, 6, 5, 4]
5. A = [2, 1, 4, 5, 6, 7, 3]
**Answer.** alternative 0: A = [2, 1, 3, 5, 6, 7, 4]
**Method (reusable).** 1. CLRS Lomuto PARTITION(A,p,r): pivot $x = A[r]$ (last element). 2. Set $i = p-1$. 3. For $j$ from $p$ to $r-1$: if $A[j] \le x$, increment $i$ and swap $A[i]$ with $A[j]$. 4. After the loop, swap $A[i+1]$ with $A[r]$ (place pivot). 5. Everything left of the pivot is $\le$ pivot, everything right is $>$ pivot.
**Worked solution.** Pivot $x = A[7] = 3$. Start $i=0$. j=1: $A[1]=6>3$ skip. j=2: $A[2]=2\le3$, $i=1$, swap $A[1],A[2]$ -> [2,6,4,5,1,7,3]. j=3: $4>3$ skip. j=4: $5>3$ skip. j=5: $A[5]=1\le3$, $i=2$, swap $A[2],A[5]$ -> [2,1,4,5,6,7,3]. j=6: $7>3$ skip. End loop: swap $A[i+1]=A[3]$ with $A[7]$ -> [2,1,3,5,6,7,4]. Result = alternative 0.

### Question 14 ÔÇö Graph algorithm trace: BFS distance (d-value) assignment order with alphabetically sorted adjacency lists.

Read the data dependencies: each cell $L(i,j)$ depends on $L(i+1,j)$ and $L(i,j-1)$. 3. Find an ordering where a cell's dependencies all live in one previously-completed "frontier" (row, column, or diagonal). Group by $d=j-i$: $L(i,i+d)$ depends on cells on diagonal $d-1$. 4. Minimum space = size of the largest frontier you must keep. 5. The frontier holds $O(n)$ values, so minimum space is $\Theta(n)$.
**Worked solution.** The subproblems are pairs $(i,j)$ with $1\le i\le j\le n$, an upper-triangular grid of $\Theta(n^2)$ cells (naive full table). But minimum space depends on dependencies. Group cells by diagonal $d=j-i$: base case is $d=0$ ($i=j$). For $d>0$, $L(i,i+d) = i\cdot L(i+1,i+d) + (i+d)\cdot L(i,i+d-1)$. Both $L(i+1,i+d)$ and $L(i,i+d-1)$ have length $d-1$, so both lie on diagonal $d-1$. Computing diagonal $d$ needs only diagonal $d-1$. Process $d=0,1,\dots,n-1$, keeping just the previous diagonal. Each diagonal holds at most $n$ entries, so peak storage is $\Theta(n)$. $L(1,n)$ is the single cell on diagonal $d=n-1$. Minimum space $= \Theta(n)$.
