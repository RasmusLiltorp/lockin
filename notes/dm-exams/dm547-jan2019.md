## Question 1 (SpĂ¸rgsmĂĄl 1) â€” Which propositions are true?

1. **Problem.** For each statement decide if it is true over the stated domain. Statements (swap these for your own):
   - 1.a $\exists n \in \mathbb{Z}: n^2 + 1 = 82$
   - 1.b $\exists n \in \mathbb{Z}: \exists k \in \mathbb{Z}: n + k = n - k$
   - 1.c $\forall n \in \mathbb{Z}: n^2 \in \mathbb{N}$
   - 1.d $\forall n \in \mathbb{Z}: \exists k \in \mathbb{N}: \sqrt{n+k} \in \mathbb{N}$
   - 1.e $\forall n \in \mathbb{N}: \forall k \in \mathbb{Z}: n \neq k$
   - 1.f $\exists n \in \mathbb{N}: 1^n \neq 1$
   - 1.g $\exists! n \in \mathbb{Z}: n^2 = 9$
   - 1.h $\exists n \in \mathbb{N}: \exists k \in \mathbb{N}: n^2 + k^2 = 17$

2. **Type.** Predicate logic â€” evaluating quantified statements (kvantorer).

3. **Method.**
   1. For $\exists$: find one witness â†’ true. For $\forall$: find one counterexample â†’ false.
   2. For $\exists!$ (exactly one): true only if there is exactly one witness; two witnesses make it false.
   3. Watch the domain ($\mathbb{Z}$ vs $\mathbb{N}$) and remember $\mathbb{N}$ here contains $0$.

4. **Worked solution.**
   - 1.a TRUE. $n=9$ (or $-9$): $81+1=82$.
   - 1.b TRUE. $n+k=n-k \iff 2k=0 \iff k=0$; pick any $n$, $k=0$.
   - 1.c TRUE. $n^2 \ge 0$ for all integers, and $0,1,4,\dots \in \mathbb{N}$.
   - 1.d TRUE. Given any $n$, choose $k$ so that $n+k$ is a perfect square $\ge 0$ (e.g. the next square $\ge n$); then $\sqrt{n+k}\in\mathbb{N}$.
   - 1.e FALSE. Counterexample $n=1\in\mathbb{N}$, $k=1\in\mathbb{Z}$ gives $n=k$.
   - 1.f FALSE. $1^n=1$ for every $n$, so no witness.
   - 1.g FALSE. $n^2=9$ has two solutions $n=3$ and $n=-3$, so "exactly one" fails.
   - 1.h TRUE. $4^2+1^2=16+1=17$.

   **Answer: 1.a, 1.b, 1.c, 1.d, 1.h are true.** (checked: Facit; checked: Python)

## Question 2 (SpĂ¸rgsmĂĄl 2) â€” Which are equivalent to $\neg(p \lor q)$?

1. **Problem.** Find every statement logically equivalent (Ă¦kvivalent) to $\neg(p \lor q)$. Candidates: 2.a $p\Rightarrow q$, 2.b $\neg p\Rightarrow q$, 2.c $p\oplus q$, 2.d $\neg p\land\neg q$, 2.e $\neg(p\Rightarrow q)$, 2.f $\neg(\neg p\Rightarrow q)$, 2.g $\neg p\land(p\oplus\neg q)$, 2.h $(\neg p\lor q)\land\neg q$.

2. **Type.** Propositional logic â€” logical equivalence (truth tables / De Morgan).

3. **Method.**
   1. Reduce the target to a canonical form. Here De Morgan: $\neg(p\lor q)\equiv \neg p\land\neg q$ â€” true only when both $p,q$ are false.
   2. Build the $2^2=4$-row truth table for the target and each candidate (swap in your own formulas).
   3. A candidate is equivalent iff its column matches the target column in all rows.

4. **Worked solution.** Target $\neg(p\lor q)$ is true only at $(p,q)=(F,F)$.
   - 2.d $\neg p\land\neg q$ â€” true only at $(F,F)$. MATCH.
   - 2.f $\neg(\neg p\Rightarrow q)$: $\neg p\Rightarrow q\equiv p\lor q$, so $2.f\equiv\neg(p\lor q)$. MATCH.
   - 2.g $\neg p\land(p\oplus\neg q)$: needs $\neg p$ (so $p=F$) and $p\oplus\neg q$ true. With $p=F$, $p\oplus\neg q=\neg q$, so need $q=F$. True only at $(F,F)$. MATCH.
   - 2.h $(\neg p\lor q)\land\neg q$: $\neg q$ forces $q=F$; then $\neg p\lor q=\neg p$ needs $p=F$. True only at $(F,F)$. MATCH.
   - 2.a, 2.b, 2.c, 2.e do not match (verified by truth table).

   **Answer: 2.d, 2.f, 2.g, 2.h.** (checked: Facit; checked: Python)

## Question 3 (SpĂ¸rgsmĂĄl 3) â€” Valid induction proofs that $3^n-1$ is even for all $n\in\mathbb{N}$

1. **Problem.** Six candidate induction arguments (induktionsbeviser) for "$3^n-1$ is even for all $n\in\mathbb{N}$". Decide which are valid.

2. **Type.** Proof technique â€” validity of mathematical induction.

3. **Method.** A valid induction proof needs:
   1. **Base case** at the smallest value the claim must hold for (here $n=0$, or any case that still covers $n=0$ via the step). If the base starts too high (e.g. $n=2$), the proof does not cover all $n\in\mathbb{N}$ â†’ invalid.
   2. **Induction step**: assume the claim for the smaller case (induktionsantagelsen) and derive it for the next. The algebra must be correct and must actually use the hypothesis.
   3. The chain of implications/equivalences must be sound; dividing in a way that leaves non-integers, or starting the step from an assumed conclusion incorrectly, breaks it.

4. **Worked solution.** Hypothesis: $3^m-1=2k$.
   - 3.a VALID. Base $3^0-1=0$ even. Step for $n\ge1$: $3^n-1=3\cdot3^{n-1}-1=3(3^{n-1}-1)+2=3\cdot2k+2=2(3k+1)$. Correct, uses hypothesis on $n-1$.
   - 3.b VALID. Same algebra written as $3^{n+1}-1=3(3^n-1)+2=2(3k+1)$, base $n=0$, step $n\ge0$. Correct.
   - 3.c VALID. Same step as 3.a; extra base case $3^1-1=2$ is harmless (still covers all $n$). Correct.
   - 3.d INVALID. Base is $3^2-1=8$ and step runs for $n\ge3$, so $n=0$ and $n=1$ are never proven. Does not cover all $\mathbb{N}$.
   - 3.e VALID. Starts from hypothesis $3^n-1=2k\iff 3^n=2k+1\iff 3^{n+1}=6k+3\Rightarrow 3^{n+1}-1=2(3k+1)$. Reversible/sound chain landing on the next case.
   - 3.f INVALID. Writes $3^n-1=\tfrac13\cdot3^{n+1}-1=\tfrac13\cdot2k=2\cdot\tfrac{k}{3}$ and claims $\tfrac{k}{3}\in\mathbb{Z}$, which is false in general. Bad algebra.

   **Answer: 3.a, 3.b, 3.c, 3.e are valid.** (checked: Facit)

## Question 4 (SpĂ¸rgsmĂĄl 4) â€” Properties of a relation

1. **Problem.** $R=\{(a,a),(a,b),(b,a),(c,c)\}$ on $\{a,b,c\}$. Decide which hold: 4.a not reflexive, 4.b symmetric, 4.c anti-symmetric, 4.d transitive, 4.e not an equivalence relation, 4.f partial order.

2. **Type.** Relations â€” checking properties (egenskaber ved relationer).

3. **Method.** For a relation on set $S$ (swap in your own pairs):
   1. **Reflexive (refleksiv):** $(x,x)\in R$ for every $x\in S$.
   2. **Symmetric (symmetrisk):** $(x,y)\in R \Rightarrow (y,x)\in R$.
   3. **Anti-symmetric:** $(x,y),(y,x)\in R \Rightarrow x=y$.
   4. **Transitive (transitiv):** $(x,y),(y,z)\in R \Rightarrow (x,z)\in R$.
   5. **Equivalence relation:** reflexive + symmetric + transitive.
   6. **Partial order (partiel ordning):** reflexive + anti-symmetric + transitive.

4. **Worked solution.**
   - Reflexive? Need $(a,a),(b,b),(c,c)$; $(b,b)$ missing â†’ not reflexive â†’ **4.a TRUE**.
   - Symmetric? $(a,b)$ and $(b,a)$ both present; $(a,a),(c,c)$ are their own mirrors â†’ **4.b TRUE**.
   - Anti-symmetric? $(a,b)$ and $(b,a)$ with $a\neq b$ â†’ fails â†’ 4.c FALSE.
   - Transitive? $(b,a)$ and $(a,b)$ require $(b,b)$, which is absent â†’ fails â†’ 4.d FALSE.
   - Equivalence? Fails reflexivity and transitivity â†’ not an equivalence â†’ **4.e TRUE**.
   - Partial order? Needs anti-symmetry (fails) â†’ 4.f FALSE.

   **Answer: 4.a, 4.b, 4.e.** (checked: Facit; checked: Python)

## Question 5 (SpĂ¸rgsmĂĄl 5) â€” Transitive closure

1. **Problem.** Give the transitive closure (transitive lukning) of $R=\{(a,b),(c,d),(d,e)\}$. Pick from the listed options.

2. **Type.** Relations â€” transitive closure.

3. **Method.**
   1. Start with all pairs in $R$.
   2. Whenever $(x,y)$ and $(y,z)$ are present, add $(x,z)$. Repeat until no new pair appears (this is the smallest transitive relation containing $R$ â€” reachability via one or more steps).
   3. Do NOT add reflexive pairs $(x,x)$ unless they arise from a cycle; transitive closure is not reflexive closure.

4. **Worked solution.** Pairs: $(a,b),(c,d),(d,e)$. The only composable pair is $(c,d)$ then $(d,e)$ â†’ add $(c,e)$. No further compositions ($a,b$ leads nowhere; $e$ leads nowhere). Result:
   $$\{(a,b),(c,d),(c,e),(d,e)\}.$$

   **Answer: 5.c.** (checked: Facit; checked: Python)
